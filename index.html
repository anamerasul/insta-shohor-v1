JavaScript is a client-side scripting language and one of the most efficient, commonly used scripting languages. The term .client-side scripting language means that it runs at the client-side( or on the client machine) inside the web-browsers, To run JavaScript chrome browser using v8 engine, Firefox using Spidermonkey, Edge using Chakra and Safari using JavaScript Core. Now a days to run JavaScript on server-side using Nodejs which is build on based on v8 engine and c++, To run JavaScript file on JavaScript engine and NodeJS every process are same,first JavaScript engine or NodeJS takes the JavaScript file and converted into the machine code so that the JavaScript run. Then its execute the code line by line which is called Synchronous behavior by default It's does not execute a function or code before the previous process is finished. There is an asynchronous behavior on JavaScript, in this behavior JavaScript runs same as Synchronous but difference is if a function or code take more time to execute the JavaScript engine or NodeJS put the function or code into a queue ,after executing all code or function then executing the code or function which is on the queue
